TestVagrantCodingTest
=============================================================================================
Description:
------------
	 It a simple maven project with added feature of cucumber and JUint. So follow the below steps for configuration.

Steps:
------
1. Import the project using the git url.
2. Open the project folder as existing maven project.
3. Run mvn clean + install.
4. Go to the following path: src/test/java/testRunners/TestRunner.java
5. Select TestRunner.java and Right Click and Select option 'JUnit Test'
6. After Successful run select root folder (testvargantcodingtest) Right Click and select option 'Refresh'
7. Updated reports will get generated for the test and you can access them by the following paths
	HTML : target/HtmlReports/report.html
	JSON : target/JSONReports/report.json
	XML	 : target/JUnitReports/report.xml
